## Overview

The Task Management system provides comprehensive functionality for creating, tracking, and organizing work items within projects. Tasks include priorities, states, assignments, and expiration dates.

## Task Data Structure

The `TblTask` model represents a task in the system:

<ParamField path="id" type="integer">
  Unique task identifier (auto-generated)
</ParamField>

<ParamField path="title" type="string" required>
  Task title (required)
</ParamField>

<ParamField path="description" type="string">
  Detailed task description (optional)
</ParamField>

<ParamField path="stateId" type="integer" required>
  Current state ID (e.g., Pending, In Progress, Completed)
</ParamField>

<ParamField path="priorityId" type="integer" required>
  Priority level ID (e.g., High, Medium, Low)
</ParamField>

<ParamField path="projectId" type="integer" required>
  Associated project ID
</ParamField>

<ParamField path="asignedId" type="integer" required>
  User ID of assigned team member
</ParamField>

<ParamField path="expirationDate" type="DateTime" required>
  Task due date
</ParamField>

<ParamField path="estimatedHours" type="integer" default="0">
  Estimated hours to complete the task
</ParamField>

<ParamField path="usercreate" type="integer">
  ID of user who created the task
</ParamField>

<ParamField path="creationDate" type="DateTime">
  Timestamp when task was created
</ParamField>

<ParamField path="modificationDate" type="DateTime">
  Timestamp of last modification
</ParamField>

<ParamField path="active" type="boolean">
  Whether the task is active (soft delete flag)
</ParamField>

## API Endpoints

```http

Authorization: Bearer {token}
```

### Create Task

Creates a new task with the specified details.

```http
POST /api/tasks
Authorization: Bearer {token}
Content-Type: application/json
```

<CodeGroup>
```javascript React Frontend
const handleTaskCreate = async (taskData) => {
  const response = await fetch('/api/tasks', {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
      'Authorization': `Bearer ${token}`
    },
    body: JSON.stringify(taskData)
  });

  const result = await response.json();
  if (result.success) {
    console.log('Task created with ID:', result.data?.id);
  }
};
```

```json Request Body
{
  "title": "Migrate database schema",
  "description": "Update schema to new format",
  "stateId": 1,
  "priorityId": 1,
  "projectId": 5,
  "asignedId": 3,
  "expirationDate": "2026-03-15T00:00:00",
  "estimatedHours": 8
}
```

```json Response
{
  "success": true,
  "message": "Task created successfully",
  "data": {
    "id": 15,
    "title": "Migrate database schema",
    "stateId": 1,
```

<Note>
The `userId` is automatically extracted from the JWT token claims, so you don't need to include it in the request body.
</Note>

### Update Task

Updates an existing task.

```http
PUT /api/tasks
Authorization: Bearer {token}
Content-Type: application/json
```

<CodeGroup>

### Delete Task

Soft deletes a task (sets `active` to false).

```http
DELETE /api/tasks/{id}
Authorization: Bearer {token}
```

<CodeGroup>
```javascript React Frontend
const handleTaskDelete = async (taskId) => {
  if (window.confirm('Are you sure you want to delete this task?')) {
    const response = await fetch(`/api/tasks/${taskId}`, {
      method: 'DELETE',
      headers: {
        'Authorization': `Bearer ${token}`
      }
    });

    const result = await response.json();
    if (result.success) {
      console.log('Task deleted successfully');
    }
  }
};
```
</CodeGroup>

<Warning>
Deletion is permanent and cannot be undone. The task's `active` flag is set to false.
</Warning>

### Priority and State Colors

The frontend implements color coding for priorities and states:

```javascript
const getPriorityColor = (priorityId) => {
  const priority = priorities.find(p => p.id === priorityId);
  const nameUpper = priority.name?.toUpperCase() || '';
  
  if (nameUpper.includes('HIGH')) return 'bg-red-50 text-red-700 border-red-200';
  if (nameUpper.includes('MEDIUM')) return 'bg-amber-50 text-amber-700 border-amber-200';
  if (nameUpper.includes('LOW')) return 'bg-green-50 text-green-700 border-green-200';
  return 'bg-gray-50 text-gray-700 border-gray-200';
};
```

## History Integration

Task operations automatically log history events:

```javascript
// Task creation logs
await logHistoryEvent(
  newTaskId,
  'CREATED',
  `Task created: "${taskData.title}"`
);

// Task updates track changes
const changes = [];
if (oldTask.title !== taskData.title) {
  changes.push(`Title: "${oldTask.title}" → "${taskData.title}"`);
}
if (oldTask.stateId !== taskData.stateId) {
  changes.push(`State: "${oldState}" → "${newState}"`);
}

await logHistoryEvent(
  taskData.id,
  'UPDATED',
  `Changes made:\n${changes.join('\n')}`
);
```

See the [History](/advanced/history) documentation for more details on tracking task changes.
